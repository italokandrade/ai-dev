# Documentacao tecnica local

Esta pasta guarda snapshots enxutos de documentacao para uso dos agentes do AI-Dev.

Regra de manutencao:

- manter apenas conteudo consultavel por `rg` e legivel por IA;
- nao guardar clones Git, `node_modules`, assets, testes, builds, configs de CI ou manifests de pacote;
- preferir Markdown, MDX, Blade docs, exemplos pequenos e definicoes de API quando a ferramenta nao trouxer docs completas no snapshot.

Conteudo atual:

- `laravel13-docs/`: paginas Markdown do Laravel 13, Laravel AI, MCP e Boost.
- `livewire4-docs/`: paginas Blade/Markdown do Livewire 4.
- `filament5-docs/`: paginas Markdown do Filament 5.
- `tailwind-docs/`: paginas MDX e guias de instalacao do Tailwind.
- `alpine-docs/`: paginas Markdown do Alpine.js.
- `animejs-docs/`: README, definicoes TypeScript de API e exemplos JavaScript pequenos do Anime.js.
